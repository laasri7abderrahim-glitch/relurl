# RELURL Chrome Extension

A Chrome browser extension for the RELURL URL shortener.

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (toggle in top-right corner)
3. Click **Load unpacked**
4. Select the `extension/` folder from this directory
5. The RELURL extension will appear in your toolbar

## Usage

1. Click the RELURL icon in the toolbar
2. The current tab's URL is automatically detected
3. Click **Shorten** to create a short link
4. Click **Copy** to copy the short URL to clipboard
5. Visit **Settings** to add your API key from relurl.com/dashboard/api-keys

## API Key

To shorten URLs, you need an API key:
- Go to https://relurl.com/dashboard/api-keys
- Generate or copy your API key
- Open the extension settings (right-click icon > Options)
- Paste your API key and save
